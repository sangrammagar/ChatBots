# 🤖 AI Chatbot using Hugging Face & Streamlit

An interactive AI chatbot built using Hugging Face Inference API and Streamlit.
This project provides a ChatGPT-like interface with real-time responses and conversation memory.

---

## 🚀 Features

* 💬 ChatGPT-like UI using Streamlit
* 🧠 Conversation memory (multi-turn chat)
* ⚡ Real-time responses using Hugging Face API
* 🎛️ Clean and simple interface
* 🔐 Secure API key using `.env`

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Hugging Face Inference API
* python-dotenv

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/chatbot-project.git
cd chatbot-project
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Create `.env` file

```
HUGGINGFACE_API_KEY=your_api_key_here
```

---

### 4️⃣ Run the app

```bash
streamlit run chatbot.py
```

---

### 5️⃣ Open in browser

```
http://localhost:8501
```

---

## 🔑 Get Hugging Face API Key

1. Go to [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
2. Create a new token
3. Paste it in `.env`
